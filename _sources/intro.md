# Installation
## From pip
```bash
pip install ectoolkits
```

## From source
One can download the source code of cp2kdata by
```bash
git clone https://github.com/robinzyb/ECToolkits.git ectoolkits
```
then use `pip` to install the module from source

```bash
cd ectoolkits
pip install .
```
